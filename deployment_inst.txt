	Deployment Instruction
	----------------------
	
Step 1:- Install Xampp cross-platform web server(localhost) fom this website according to the OS fromn given link 
	  or u can find it in res folder of the zip itself.
	  link:- https://www.apachefriends.org/download.html
	  
Step 2:- copy the dist folder into the htdocs folder of Xampp.

	  	
Step 3:- Import the database in which the data will be stored when someone will register. While importing make sure that the database name is 'event'.

Step 4:- Open dist folder open connect.php file and change the id password. 

Step 5:- Now go to your browser and type http://localhost/dist/

Step 6:- Fill the form and the data will be fetched and stored into database.

Step 7:- You can go to admin login to check all details. Some id and passwords are: user- root, pass-event; user- root, password-demo.
